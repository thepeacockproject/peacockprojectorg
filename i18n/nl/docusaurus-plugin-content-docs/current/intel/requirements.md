### Niet-ondersteunde games en platforms

-   Elk ander platform dat niet wordt vermeld in het gedeelte “Ondersteund”, zoals consoleplatforms, worden niet ondersteund.
-   Demoversies van Hitman 3 worden niet ondersteund.
-   Game Pass of Microsoft PC-editie van Hitman 3 wordt niet ondersteund.
-   **Elke gekraakte versie van Hitman 1-3 wordt niet ondersteund en je wordt verbannen uit de Peacock Discord als je vraagt hoe je een gekraakte versie kunt gebruiken.**

## Ondersteunde besturingssystemen

### Server

De server is het gedeelte dat verantwoordelijk is voor het afhandelen van de verzoeken van het spel. Het is mobiel en hoeft niet eens op dezelfde machine te draaien als het spel.

-   Een platform dat wordt ondersteund door Node.js, dus Windows 8.1 of hoger, of een recente Linux/macOS/FreeBSD-installatie.
-   We hebben alternatieven beschikbaar als je niet aan deze eisen kunt voldoen, vraag hiernaar in de Discord.
-   Bekijk de Linux NodeJS installatiegids: https://thepeacockproject.org/wiki/guides/linux-setup/

### Patcher

De patcher vertelt het spel om daadwerkelijk verbinding te maken met Peacock en moet zich op dezelfde machine bevinden als het spel.

-   Windows 7 of hoger

### Samenvatting

Windows 10 en 11 zijn de best ondersteunde besturingssystemen, maar Windows 7 of 8 en Linux zouden ook moeten werken, met extra probleemoplossing. **Vraag het in de [Discord](https://thepeacockproject.org/discord) in geval van twijfel.**

:::warning Windows 7 ondersteuning

Windows 7 wordt niet langer ondersteund door Microsoft of Node.js en daarom kunnen we niet garanderen dat het een bugvrije ervaring zal zijn.

Upgrade naar een nieuwere versie van Windows voor beveiligingsupdates en een stabielere ervaring.

Sommige eerdere Hitman-games (2016 en 2) ondersteunen Windows 7, maar je zult de server op een andere machine of virtuele machine moeten hosten om het te laten werken.

:::
