title: 進度轉移
---

Peacock 支援從官方《殺手世界》遊戲伺服器轉移您的進度。 這將涵蓋您在 Peacock 上的所有當前進度。

我們目前僅支援相同的遊戲和平台，例如，您只能從 Hitman 3 Steam → Hitman 3 Steam。
計劃支援跨遊戲進度轉移。

## 我該怎麼做？

1. 首先，啟動 Peacock。 然後將 Peacock 與遊戲連接起來。 建議您現在關閉遊戲。

2. 導航到 Peacock 的 Web UI，大多數用戶可以[點擊這裡](http://localhost/)訪問它。
   如果 Peacock 在不同的連接埠上執行，您可以在 Web 瀏覽器中導航至`http://localhost:PORT/`。

3. 然後，轉到頁面頂部導覽列上的**進度轉移**按鈕。
   選擇您想要轉移進度的帳戶。

4. 您可以在此處查看上次轉移此帳戶進度的時間（如果適用）。
   現在按下綠色的**立即同步**按鈕。

5. 將會彈出一個窗口，**請仔細閱讀**——這是您選擇退出的最後機會。

您現在將看到一個彈出框。 如果是**綠色**，則表示已成功完成！
如果是**紅色**，則表示失敗，請閱讀下文以了解一些常見問題及其解決方法。

## 常見問題

### 無法取得官方認證數據

**如果您沒有先連接到 PEACOCK**：戴上老花眼鏡返回步驟 1。

如果您這樣做了：您確定嗎？ 您看到 Peacock 儀表板了嗎？ 返回步驟 1 並確保遵循它們。

\*\*您擁有這款遊戲嗎？ \*\*如果不是，那麼它將無法運作。

### 同步官方數據失敗

確保您遵循了上述所有步驟。 \*\*確保您擁有該遊戲。 \*\*
如果您仍然遇到此問題，請[提交問題](https://github.com/thepeacockproject/Peacock/issues/new/choose)或[加入我們的Discord](https://discord.gg/F8qQTfnajw)。
